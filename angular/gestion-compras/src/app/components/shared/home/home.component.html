Sistema de Gestión Compras

Software pensado para automatizar y optimizar.

Esta aplicación de gestión de compras es una herramienta pensada para automatizar y optimizar el proceso de adquisición de bienes y servicios en una empresa.

El sistema permite:

  • Dar de alta, modificar y/o eliminar proveedores.
  • Crear, modificar y eliminar productos y o servicios, con su respectiva información.
  • Generar órdenes de compra para un proveedor con sus respectivos productos.
